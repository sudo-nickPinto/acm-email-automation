# Original README (v1 Archive)

This file preserves the original v1 README before the v2 multi-source rewrite. It's kept as a historical reference only.

The v1 project was a single-source ACM TechNews email automation tool that used macOS `launchd` for scheduling. v2 expanded it to support multiple newspaper sources with an interactive terminal setup wizard.

See the [current README](../README.md) for the active project documentation.
