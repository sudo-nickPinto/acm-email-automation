# Glossary

## Protocols & Standards

| Term | Definition |
|------|-----------|
| **RSS** | Really Simple Syndication — an XML format for publishing article lists. Each `<item>` has a title, link, description, and pubDate. |
| **RSS 2.0** | The version used by all sources in this project. Structure: `<rss> → <channel> → <item>*` |
| **SMTP** | Simple Mail Transfer Protocol — the standard for sending email between servers. |
| **STARTTLS** | An SMTP extension that upgrades a plain-text connection to encrypted TLS. Used on port 587. |
| **TLS** | Transport Layer Security — encrypts data in transit. Prevents eavesdropping on credentials and email content. |
| **MIME** | Multipurpose Internet Mail Extensions — format for email bodies. `multipart/alternative` contains both plain-text and HTML versions. |
| **RFC 822** | The date format used in RSS feeds: `"Mon, 25 Mar 2026 14:30:00 GMT"` |

## Security

| Term | Definition |
|------|-----------|
| **App Password** | A 16-character, letters-only password generated by Gmail for third-party apps. Requires 2FA to be enabled. |
| **2FA** | Two-Factor Authentication — requires a second verification method (phone/app) beyond password. |
| **SHA-256** | A cryptographic hash function producing a 256-bit (64 hex char) digest. Used for checksum verification and dedup. |
| **Checksum** | A hash digest used to verify file integrity. If one bit changes, the hash changes completely. |
| **XSS** | Cross-Site Scripting — injecting malicious scripts via untrusted input. Prevented by HTML escaping. |
| **chmod 600** | Unix file permission: owner read+write, no access for group or others. Applied to `.env`. |

## Python

| Term | Definition |
|------|-----------|
| **venv** | Virtual environment — an isolated Python installation with its own packages. Prevents conflicts with system Python. |
| **dataclass** | A Python decorator (`@dataclass`) that auto-generates `__init__`, `__repr__`, etc. for classes with typed fields. |
| **ElementTree** | Python's stdlib XML parser. Parses XML into a tree of `Element` objects. |
| **python-dotenv** | Library that reads `.env` files and loads key=value pairs into `os.environ`. |
| **importlib.reload()** | Re-executes a module's code. Used by the CLI to pick up `.env` changes without restarting. |

## Operating System

| Term | Definition |
|------|-----------|
| **LaunchAgent** | macOS user-level scheduled task. A plist XML file in `~/Library/LaunchAgents/` managed by `launchd`. |
| **launchd** | macOS process and service manager (similar to systemd on Linux). |
| **cron** | Unix/Linux scheduler daemon. Reads per-user `crontab` files to run commands at specified times. |
| **crontab** | The configuration file for cron. Each line: `minute hour day month weekday command`. |
| **Task Scheduler** | Windows built-in scheduling service. Managed via `schtasks.exe` CLI or the GUI. |
| **schtasks.exe** | Windows command-line tool for creating/querying/deleting scheduled tasks. |

## Project-Specific

| Term | Definition |
|------|-----------|
| **NewsSource** | Dataclass in `sources.py` representing a newspaper (key, name, description, RSS URL, max articles). |
| **Article** | Dataclass in `scraper.py` representing a single news article (title, description, link, source name, pub date). |
| **SourceResult** | Dataclass in `scraper.py` wrapping a source's articles and/or error message. |
| **SCHEDULE_LABEL** | `"com.newsdigest.daily"` — the identifier used across all three OS scheduling backends. |
| **Edition hash** | SHA-256 of `date + all titles`. Stored in `.last_sent` to prevent duplicate emails. |
| **Dry run** | `--dry-run` flag: prints the email preview to the terminal without sending or saving state. |
